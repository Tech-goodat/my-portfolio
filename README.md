# Felix Kiprotich Portfolio

A modern, responsive, and animated portfolio website for Felix Kiprotich ‚Äî Fullstack Software and AI Engineer. Built with [Next.js 13+](https://nextjs.org/), [React](https://react.dev/), [Tailwind CSS](https://tailwindcss.com/), and [Framer Motion](https://www.framer.com/motion/) for sleek UI and smooth animations.

---

## üöÄ Features

- **Responsive Design:** Fully mobile-friendly and adapts to all screen sizes.
- **Dark/Light Mode:** Uses system theme with smooth color transitions.
- **Animated UI:** Onboarding and scroll animations with Framer Motion.
- **Floating Mobile Navbar:** Modern, icon-only bottom navigation for mobile.
- **Projects Showcase:** Modern cards with live and code links, animated on load.
- **Testimonials:** Real testimonials with avatars and animated appearance.
- **Education & Skills:** Responsive, visually appealing layouts with relevant icons.
- **Contact Page:** Minimal, two-column layout with clickable contact links and working email form.
- **Download CV:** Downloadable resume from the public folder.
- **Accessible:** Keyboard navigable and semantic HTML.

## üì¶ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Tech-goodat/my-portfolio.git
cd my-portfolio
```

### 2. Install dependencies

```bash
npm install
# or
yarn install
```

### 3. Run the development server

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## üñºÔ∏è Customization

- **Profile Info:**  
  Update your name, bio, and social links in `/app/page.tsx` and `/app/components/Footer.tsx`.

- **Projects:**  
  Edit `/app/projects/page.tsx` to add your own projects, images, and links.

- **Skills & Education:**  
  Update `/app/skills/page.tsx` and `/app/education/page.tsx` with your details.

- **Testimonials:**  
  Edit `/app/testimonials/page.tsx` to add real testimonials and images.

- **Resume:**  
  Replace `/public/resume.pdf` with your own CV.

- **Contact:**  
  Update your email, phone, WhatsApp, and social links in `/app/contact/page.tsx`.
